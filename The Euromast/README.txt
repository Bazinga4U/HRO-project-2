
For this project we used the following programms,
Make sure you have these versions of the programs installed for the best expierence.

- Python 3.4.4
- Pygame 1.9.2 (for python 3.4.4)
- Psycopg2 2.6.2 (For python 3.4.X)
- PgAdmin 4 
- Postgresql 9.6.1
- Microsoft Visual Studio 2015
- Jet Brains Pycharm Community Edition 2016.2.2

Before launching the game make sure you made the database in preferably pgAdmin 4, 
with the given databasequery.sql or databasequery.txt. 
When you created the databse, the scoreboard will function and the game can be finished.

To launch the game you need to use the "Game versie 0.1.sln"

You can use one of these programs to launch the given file above.
- Microsoft Visual Studio 2015
- Jet Brains Pycharm Community Edition 2016.2.2



Have fun and enjoy the game!
